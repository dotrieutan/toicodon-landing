# Tôi cô đơn landing page

Static landing page for the Tôi cô đơn companionship app.

## Preview locally

Open `index.html` directly in a browser, or run any small static server from this folder:

```bash
python3 -m http.server 5174
```

Then visit `http://localhost:5174`.

## Publish with GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Set the source to the main branch and the repository root.
4. Save and wait for GitHub Pages to publish the site.
